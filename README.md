<h1>Bot de Envio e Criação de Lotes da Tesouraria</h1>

<p>Este é um bot desenvolvido para facilitar o envio e a criação de lotes da tesouraria. Ele foi construído utilizando apenas Python e suas bibliotecas, aproveitando as capacidades da linguagem para manipular planilhas, pastas no computador e enviar e-mails de forma automatizada.</p>

<h1>Funcionamento:</h1>

<p>O sistema é bastante simples e intuitivo. Ao executá-lo, você será apresentado à interface visual do bot, que consiste em uma tela inicial com um texto explicativo sobre suas funcionalidades, além de duas opções principais:</p>

<h2>Tela Inicial:</h2>

<img src="https://github.com/GabriellpTV/PedidosAprovacoes/assets/138072118/e6910d3d-bba6-452a-a82d-183e5dd4352d" width="600">

<p>Nesta tela, você encontrará uma interface amigável que introduz as capacidades do bot:</p>

<h2>Configurações:</h2>

<img src="https://github.com/GabriellpTV/PedidosAprovacoes/assets/138072118/17d6dfc9-22de-4725-b797-d6928c0227d4" width="600">

<p>No menu de configurações, você terá acesso a duas funcionalidades: editar e-mail e guia de uso.</p>
<h3>Guia de Uso:</h3>
<p>A opção "Guia de Uso" traz o usuário para este arquivo que você está lendo agora, onde ele pode visualizar as funcionalidades do bot e entender como resolver possíveis problemas no uso diário.</p>
<h3>Editar E-mail:</h3>

<img src="https://github.com/GabriellpTV/PedidosAprovacoes/assets/138072118/0e380c44-a350-4c6e-940b-ab74d82ae7d5" width="600">

<p>Ao acessar a opção de edição de e-mail, uma caixa de diálogo será aberta, permitindo ao usuário selecionar o modelo de e-mail desejado para edição. Ao clicar, ele será redirecionado para o Outlook, onde poderá realizar as alterações necessárias.</p>

<img src="https://github.com/GabriellpTV/PedidosAprovacoes/assets/138072118/468f08a2-8e09-4955-b476-67bbedcd5d5e" width="600">

<p>Ao acessar a opção de edição de e-mail, uma caixa de diálogo será aberta, permitindo ao usuário selecionar o modelo de e-mail desejado para edição. Em seguida, o Outlook será aberto com o modelo de e-mail selecionado, onde o usuário poderá fazer todas as alterações necessárias.</p>
<p>Após fazer as alterações desejadas, o usuário deve clicar em "Salvar" e, em seguida, fechar o arquivo. É importante observar que os campos de envio, como Cópia, Assunto e Para, não devem ser preenchidos neste momento.</p>

<h2>Inserir Planilha:</h2>

<img src="https://github.com/GabriellpTV/PedidosAprovacoes/assets/138072118/8c23d541-870e-425f-be46-5ada74163a93" width="600">

<p>Ao selecionar esta opção, você será direcionado para a tela de seleção de planilhas. Aqui, é importante escolher a planilha de pedidos de compra JÁ ATUALIZADA, para que o envio de e-mails seja realizado de forma correta.</p>

<img src="https://github.com/GabriellpTV/PedidosAprovacoes/assets/138072118/e091c34d-cbf5-46dc-95d0-4d93aa1ef2f0" width="600">

<p>Após selecionar a planilha, você deverá confirmar se deseja prosseguir com o processo. Na tela, será exibida a planilha selecionada e, caso haja algum erro, você poderá corrigi-lo utilizando o botão "Inserir Novamente". Se tudo estiver correto, basta clicar em Continuar.</p>

<img src="https://github.com/GabriellpTV/PedidosAprovacoes/assets/138072118/5aee6ce7-4278-4bd4-93e8-c964879f3b5a" width="600">

<p>Caso você se depare com uma mensagem informando que a janela não está respondendo, não há necessidade de se preocupar. Isso ocorre devido ao processo de filtragem da planilha de pedidos e à criação dos lotes com pedidos de compra por aprovador de compra.</p>

<h2>Enviar E-Mails:</h2>

<img src="https://github.com/GabriellpTV/PedidosAprovacoes/assets/138072118/3e896e35-1f90-4daf-866f-5cc28d507ff2" width="600">

<p>Após os lotes serem criados, eles serão exibidos na tela junto com dois botões: um para voltar e outro para enviar.</p>

<h3>Voltar</h3>

<p>Os lotes anteriormente criados serão excluídos e você será redirecionado para a tela inicial do aplicativo, caso deseje iniciar o processo novamente.</p>

<h3>Enviar</h3>

<img src="https://github.com/GabriellpTV/PedidosAprovacoes/assets/138072118/0e380c44-a350-4c6e-940b-ab74d82ae7d5" width="600">

<p>Você será solicitado a selecionar o modelo de e-mail para o envio dos e-mails. Após selecionar, todos os lotes serão enviados pelo seu respectivo aprovador.</p>

<img src="https://github.com/GabriellpTV/PedidosAprovacoes/assets/138072118/d08a35b9-c1f6-40b8-9be8-3f5de09dbc48" width="600">

<p>Se uma mensagem como a mencionada acima aparecer, basta clicar em "Enviar" mesmo assim. Isso ocorre devido ao envio dos e-mails ser feito via script.</p>

<img src="https://github.com/GabriellpTV/PedidosAprovacoes/assets/138072118/1b14a453-bf6f-4f4c-a783-bb803f3a511f" width="600">

<p>Com isso, os e-mails serão enviados e aparecerão na sua caixa de entrada como enviados, seguindo o modelo especificado.</p>

<img src="https://github.com/GabriellpTV/PedidosAprovacoes/assets/138072118/86e52f67-99fd-48f9-a84d-4a4e71245d59" width="600">

<p>Também será exibida uma mensagem na tela informando que os e-mails foram enviados com sucesso, juntamente com um botão "Concluir". Ao clicar neste botão, os lotes do dia serão enviados para a pasta Ano > Mês para armazenamento.</p>

<img src="https://github.com/GabriellpTV/PedidosAprovacoes/assets/138072118/0305072e-c77b-4737-a8ab-cb13ff6796e0" width="600">

<p>Essa foi a descrição do processo. Os e-mails foram enviados com sucesso!!!</p>
